na'ploqqatijig

Recordings:
  • Recording by dmm
  • Recording by ewm
  • Recording by jnw
Translation: They (dual) go in a single file
Part of Speech: verb animate intransitive
Meanings:
  • go single file
  • follow one another in straight line
Example of word used in a sentence:
  • Text: Mijua'ji'jg na'ploqqatultijig pisgwita'jig nujigina'muo'guom.
    Translation: The children go in a single file as they go into the school.
    Recording: Recording by jnw
Pronunciation Guide: naa路b锚路lohk路hka路ti路jik

Alternate Grammatical Forms:
  • na'ploqqatultijig -- They go in single file -- (first person plural exclusive animate)
