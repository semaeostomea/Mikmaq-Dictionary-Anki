esnpitgmatg

Recordings:
  • Recording by dmm
  • Recording by ewm
  • Recording by jnw
Translation: He/she packs it down while he/she loads it
Part of Speech: verb inanimate transitive
Meanings:
  • pack down while loading
Example of word used in a sentence:
  • Text: Etliesnpitgmatg sa'qawei ugpesgeweim.
    Translation: He/she is packing down his/her old gun while he/she is loading it.
    Recording: Recording by jnw 🔉
Pronunciation Guide: e·sên·bid·gêm·atk

Alternate Grammatical Forms:
  • esnpitgmatmegl -- we packed it down while loading it -- (third person plural inanimate)
Document References:
  • Pacifique Dictionary Manuscript, Page Image #172
    Transcription:esnepitgematem, le charger

    Source as entry:esnpitgmatm, I load it
    Normalized source as entry:esnpitgmatg, he/she/it loads it
