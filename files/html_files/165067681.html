nigtua'latl

Recordings:
  • Recording by dmm
  • Recording by ewm
  • Recording by jnw
Translation: He/she separates him/her from the rest
Part of Speech: verb animate transitive
Meanings:
  • separate from
Example of word used in a sentence:
  • Text: Nigtua'latl na te'sipowl ta'n pewalapnn.
    Translation: He/she separates that horse he/she wanted.
    Recording: Recording by jnw 🔉
Pronunciation Guide: nik·tu·aa·la·dêl

Alternate Grammatical Forms:
  • nigtua'l'g -- I separate him/her from the rest -- (first person singular animate subject, third person singular animate object)
  • niguta'lul -- I separate you from the rest -- (first person singular animate subject, second person singular animate object)
  • nigtua'lit -- He/she separates me from the rest -- (third person singular animate subject, first person singular animate object)

Related entries for category "seperate"

  • nasgoqwa'latl : split (by hand)
  • nigtua'latl : separate from
  • nigtua'tijig : separate / go separate ways
  • nigtua'toq : separate
  • punaltijig : leave each other / separate / divorce
