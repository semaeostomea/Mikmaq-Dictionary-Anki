poqjiws'ggewe'g

Recordings:
  • Recording by dmm
  • Recording by ewm
  • Recording by jnw
Translation: He/she starts to laugh
Part of Speech: verb animate intransitive
Meanings:
  • start to laugh
Example of word used in a sentence:
  • Text: Mu gis'punelmigw newt poqjiws'ggewe'geg.
    Translation: He/she couldn't stop laughing once he/she started.
    Recording: Recording by jnw 🔉
Pronunciation Guide: bohk·chiw·sêk·ke·weeg

Alternate Grammatical Forms:
  • poqjiws'ggewei -- I start to laugh -- (first person singular animate)
  • poqjiws'ggeweieg -- We start to laugh -- (first person dual exclusive animate)
  • poqjiws'ggewo'ltieg -- We start to laugh -- (first person plural exclusive animate)
