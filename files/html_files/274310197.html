la'pugtaqaniget

Recordings:
  • Recording by dmm
  • Recording by ewm
  • Recording by jnw
Translation: He/she sets snares
Part of Speech: verb animate intransitive
Meanings:
  • set snares
Example of word used in a sentence:
  • Text: Nipugtug eliei aq najila'pugtaqanigei.
    Translation: I'm going to the woods to set some snares.
    Recording: Recording by jnw 🔉
Pronunciation Guide: laa·puk·ta·hga·ni·get

Alternate Grammatical Forms:
  • la'pugtaqanigei -- I set snares -- (first person singular animate)
  • la'pugtaqanigeieg -- We set snares -- (first person dual exclusive animate)
  • la'pugtaqaniga'tieg -- We set snares -- (first person plural exclusive animate)
